Configuration Required

Set up a Google Spreadsheet to use the live data viewer.

  1. Create or open a Google Spreadsheet
  2. Go to File → Share → Publish to web and publish it
  3. Click Share and set access to Anyone with the link (Viewer)
  4. Copy the spreadsheet ID from the URL
  5. Set SPREADSHEET_ID in rndlivedata.config.json
  6. Set SHEET_NAME to the tab name you want to display
